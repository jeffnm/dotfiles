# dotfiles

## Configurations for Linux and MacOS

* .vimrc
* .zshrc
* .tmux.conf

### .vimrc

Uses VimPlug for plugin manager

### .zshrc

Uses oh-my-zsh and zinit

### .tmux.conf

Remaps the prefix and use vi mode. Uses TMUX plugin manager.
